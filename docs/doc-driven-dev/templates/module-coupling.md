## N. 模块依赖关系图

```
         ┌──────────┐
         │  模块A   │ ← 顶层
         └────┬─────┘
              │
     ┌────────┴────────┐
     │                 │
     ▼                 ▼
 ┌────────┐      ┌────────┐
 │ 模块B  │─────→│ 模块C  │
 └────────┘      └────────┘
```

**关键原则**：
- 谁依赖谁
- 谁不知道谁的存在
